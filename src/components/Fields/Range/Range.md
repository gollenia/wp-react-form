# `Range`

Slider-like input based on native `input type="range"`.

## Use when

- a continuous or small numeric range should be selected

## Important props

- `value`
- `onChange`
- `min`
- `max`
- `hasTicks`
- `hasLabels`

## Notes

- Semantically, this is a range/slider field, not a classic number input.
- The former `Number` field was renamed internally to `Range`.
- `numberpicker` is currently still a legacy alias for this field.
