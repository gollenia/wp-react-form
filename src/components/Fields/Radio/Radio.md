# `Radio`

Radio group for exclusive selection.

## Use when

- all options should stay visible
- exactly one value is chosen from a small set

## Important props

- `options`
- `value`
- `onChange`
- `label`
- `required`
- `error`

## Notes

- `options` can be `string[]` or `Record<string, string>`.
- For a more compact presentation or many options, prefer `Select` or `Combobox`.
